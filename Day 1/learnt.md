## Things to remember:

- pop() &rarr; by default removes the last element (-1 index means last)
- insert() is index based insert and append() is always inserts in end
- remove() &rarr; removes only the first occurence of the element specified
- reverse() &rarr; in-place function, meaning it returns "None" so use a separate line for this action
- sort() &rarr; similar to reverse(), it sorts the list "in-place"
- write_index &rarr; method for in_place_removing_duplicates
- list comprehension like &rarr; **[candidate_element iteration condition]**
